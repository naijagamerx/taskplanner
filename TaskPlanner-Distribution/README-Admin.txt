# Task Planner Admin Dashboard v1.0.0

## Purpose
This is the administrative dashboard for managing Task Planner licenses.

## Features
- Generate new license keys
- View all generated licenses
- Copy license keys to clipboard
- Delete/revoke licenses
- Hardware ID management

## Usage
1. Run TaskPlanner-Admin-v1.0.0.exe
2. Use the interface to manage licenses
3. Distribute license keys to users

## Security
- All license data is encrypted
- Hardware fingerprinting prevents unauthorized use
- Secure key generation algorithms

## Files
- TaskPlanner-v1.0.0.exe: Main application
- TaskPlanner-Admin-v1.0.0.exe: Admin dashboard
