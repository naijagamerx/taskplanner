# Task Planner v1.0.0

## Installation
1. Simply run TaskPlanner-v1.0.0.exe
2. The application will create necessary files in your AppData directory
3. All configuration is handled automatically

## Features
- Task management with countdown notifications
- Calendar view and analytics
- Persistent background notifications
- Licensing system
- Enhanced UI with themes and shortcuts
